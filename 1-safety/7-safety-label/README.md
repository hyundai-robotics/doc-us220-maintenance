# 1.7. 안전 라벨

로봇 본체에는 로봇 사용 간 발생할 수 있는 안전사고를 예방하기 위하여 하기 라벨들과 같은 안전 명판이 붙어 있습니다. 이를 불필요하게 교체하거나 제거하지 마십시오. 로봇에 부착된 명판, 경고 표시, 안전 기호, 명칭 표시, 전선 마크 등의 위치를 옮기거나 페인트칠 및 커버를 씌워 가리는 행위 등 안전 라벨에 손상을 주는 일체의 행위를 금합니다.


{% hint style="warning" %}
로봇에 부착된 명판, 경고 표시, 안전 기호, 명칭 표시, 전선 마크 등의 위치를 옮기거나 페인트칠 및 커버를 씌워 가리는 행위 등 안전 라벨에 손상을 주는 일체의 행위를 금합니다.
{% endhint %}

{% endhint %}
