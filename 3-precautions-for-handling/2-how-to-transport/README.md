# 3.2. 운반 방법

로봇 운반은 크레인 또는 지게차를 이용하여 운반 할 수 있습니다. 로봇을 이동 할 경우, 반드시 하기 표와 같은 로봇 자세로 설정 한 다음, 아이볼트(Eye bolt) 및 지정된 운반장비를 이용하여 운반 바랍니다. 로봇 운반 자세는 다음과 같습니다.
